# 康复训练计时器

这是一个纯静态网页，可免费部署到 GitHub Pages。无需服务器或数据库；训练设置仅保存于当前手机浏览器。

## 发布到 GitHub Pages

1. 在 [GitHub](https://github.com) 注册或登录账号，选择 **New repository**。
2. 仓库名称填写 `recovery-timer`，选择 **Public**，点击 **Create repository**。
3. 点击 **Add file → Upload files**，将本文件夹中的全部文件上传（不要上传父文件夹本身）。点击 **Commit changes**。
4. 打开仓库的 **Settings → Pages**。在 **Build and deployment** 中，`Source` 选 **Deploy from a branch**；`Branch` 选 `main` 和 `/ (root)`，点击 **Save**。
5. 等待约一分钟，刷新同一页面，顶部会出现网址：`https://你的用户名.github.io/recovery-timer/`。
6. 在 iPhone 的 Safari 打开这个网址，点底部“分享”按钮，选择“添加到主屏幕”。

首次打开时需要联网；之后网页会缓存，可在无网时打开。训练时请让网页保持在前台，避免 iPhone 锁屏或切换应用造成语音与计时暂停。
